Use an external database / MariaDB

The connection is set by environment variables (it can’t be edited here, since the app needs the database before it can read any saved setting). To point at an external server, set these on the app service and remove the bundled db service from docker-compose.yml:

DB_HOST=your-db-host
DB_PORT=3306
DB_NAME=priceorbit
DB_USER=priceorbit
DB_PASSWORD=your-password

The database is always its own container (MySQL 8 or MariaDB 11); only the app image is bundled.
